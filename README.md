# Bbang-psych-test
# [🍞 나는 무슨 빵일까?]

## 소개
[빵집에서 빵을 고르는 순간의 선택으로
AI가 당신의 성격과 인간관계 스타일을 빵에 비유해 알려주는 심리테스트 앱입니다 🥐✨]

## 주요 기능
- 🍞 빵집 상황 기반 5문항 심리테스트
- 🤖 **OpenAI(gpt-4o-mini)**를 활용한 AI 성격 분석
- 💞 궁합이 잘 맞는 빵 유형과 이유 제공
- 🐣 결과에 어울리는 귀여운 대표 대사 출력
- 📋 결과를 클립보드로 바로 공유 가능

## 사용 방법
1. 링크 접속
2. 질문에 답변
3. AI 결과 확인
